# Racha Conta PWA

Um aplicativo web progressivo (PWA) leve, focado em agilidade e design limpo (Vanilla HTML, CSS Tailwind, Javascript) para dividir contas de bares, restaurantes e rolês com facilidade, incluindo cálculo de gorjeta integrado e compartilhamento de cobranças via WhatsApp com o seu PIX.

## Funcionalidades
- **Cálculo Rápido:** Adicione o valor da conta e a gorjeta desejada.
- **Divisão com Amigos:** Selecione da sua lista de amigos quem vai rachar a conta na hora.
- **Gerador de PIX/Cobrança:** Gera automaticamente textos para o WhatsApp de "Joãozinho deve a Você R$ 50,00" com a sua chave PIX.
- **Histórico Local:** Guarda um histórico mensal das suas contas rachadas usando o LocalStorage do celular. Nenhuma conta, senha ou envio de dados para servidores é necessário. Privacidade total.
- **Configurações Pessoais:** Salve amigos frequentes e sua chave PIX para uso recorrente.
- **PWA Ready:** Pode ser instalado como aplicativo diretamente pelo navegador do celular.

## Como Executar
Basta hospedar os arquivos estáticos (por exemplo, no **GitHub Pages**, Vercel ou Netlify) ou abrir o `history.html` (ou futuramente o `index.html`) diretamente no navegador através de um Live Server.

## Tecnologias
- Vanilla Javascript
- HTML5
- Tailwind CSS (via CDN)
- Gerado e refinado com Google/Stitch MCP e Gemini.
